# City Chain

A two-player geography word game. Each player must name a city that starts with the last letter of the previous city — all against a countdown timer. First player to run out of time loses.

## Game Modes

- **Local** — pass-and-play on one device
- **vs Bot** — play against an AI opponent *(coming soon)*
- **Practice** — solo mode, no timer *(coming soon)*
- **Play a Friend** — shareable private link *(coming soon)*
- **Online** — random matchmaking *(coming soon)*

## Features

- Live world map with pins for every city in the chain
- Per-turn countdown timer with chess-clock style layout
- 3-2-1-GO! countdown before each game starts
- Give up / forfeit with a dedicated end screen
- Cities validated against a real dataset
